## DEV

you need an .env file with this params
```
{
  "SECRET": "<SECRET>",
  "MONGODB_URI": "<MONGODB_URI>",
  "PORT": "<PORT>",
  "WEB_CONCURRENCY": <WEB_CONCURRENCY>,
  "IMGUR_USERNAME": "<IMGUR_USERNAME>",
  "IMGUR_PASSWORD": "<IMGUR_PASSWORD>",
  "IMGUR_CLIENTID": "<IMGUR_CLIENTID>"
}

```

to run

```
npm install
npm start
ng serve
```
